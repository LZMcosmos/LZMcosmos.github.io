3 SQL中的聚合、分组和排序

1) 聚合函数

SQL中用于汇总的函数叫做聚合函数。以下五个是最常用的聚合函数：

  • COUNT：计算表中的记录数（行数）
  • SUM：计算表中数值列中数据的合计值
  • AVG：计算表中数值列中数据的平均值
  • MAX：求出表中任意列中数据的最大值
  • MIN：求出表中任意列中数据的最小值

例子：

关于聚合函数注意的点：

  • COUNT函数的结果根据参数的不同而不同。COUNT(*)会得到包含NULL的数据行数，而COUNT(<列名>)会得到NULL之外的数据行数。

  • 聚合函数会将NULL排除在外。但COUNT(*)例外，并不会排除NULL。

  • MAX/MIN函数几乎适用于所有数据类型的列。SUM/AVG函数只适用于数值类型的列。

  • 在聚合函数的参数中使用DISTINCT，可以剔除重复数据。

2) 分组子句

查询时会对整个表的数据进行处理，如果需要对不同的字段分组进行时，可以利用GROUP BY子句。在GROUP BY子句中指定的列称为聚合键或者分组列。如果聚合键中包含NULL值，分组时会将NULL作为一组特殊数据进行处理

例子： 注意：GROUP BY的子句放置的位置有要求，不合适的位置会导致SQL无法正常执行，简单的查询如：1 SELECT → 2. FROM → 3. WHERE → 4. GROUP BY。当查询语句更为复杂的时候就要理清这次的查询逻辑把分组子句放到合适的位置。

聚合函数和GROUP BY子句有关的常见错误

常见错误1：在select子句中书写了多余的列。 在使用聚合函数时，select子句中的元素有严格的限制。实际上，在使用聚合函数时，select子句中只能存在以下三种元素：

①常数；

②聚合函数；

③GROUP BY子句中指定的列名/字段（也就是聚合键）。

常见错误2：在GROUP BY子句中写了列的别名。

排序的结果默认为升序排列ASC，如果想要降序排列的话使用关键字DESC。
